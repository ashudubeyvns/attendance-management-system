✨ Features

Everything you need to manage academic workflows efficiently

🔐

Secure Login

Role-based authentication with multiple user credentials and encrypted password storage.

📊

Dashboard

Centralized home screen with quick-action module cards and real-time statistics.

✅

Mark Attendance

Daily attendance with Present/Absent toggle per student. Supports lectures & practicals.

📝

Assignments

Upload marks, manage max marks, update records with subject-wise sections.

📋

Sessionals

Enter Sessional-1, 2, 3 marks with auto-calculated totals and Pass/Fail status.

🔑

Change Password

Secure password update with validation checks and confirmation.

📱

Responsive UI

Single-window scrollable interface with CardLayout navigation across all modules.

🎓

Multi-Course

BCA, MCA, B.Tech CSE/IT, B.Sc CS, M.Sc CS with semester-wise subjects.

🚀 Getting Started

Ready to use it? Here's how

1

Download the JAR

Grab the latest release from GitHub Releases or Actions Artifacts.

2

Run it

Make sure you have JDK 26+ installed, then run:

java -jar AttendanceMgt.jar
3

Or build from source

git clone https://github.com/ashudubeyvns/attendance-management-system.git
cd attendance-management-system/AttendanceMgt_1 && javac -d build/classes src/attendancemgt/*.java src/login_frame.java && java -cp build/classes attendancemgt.AttendanceMgt
4
